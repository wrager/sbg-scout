# v0.10.0

## Ребрендинг: SBG Scout

- Приложение переименовано из SBG Userscripts в SBG Scout
- Новая иконка приложения
- Переименован пакет: sbguserscripts → sbgscout

## Бандлированный скрипт SVP

- SVP предустановлен в APK
- Если пользователь удалил бандлированный скрипт, он не переустанавливается

## Загрузка скрипта из файла

- Добавлена возможность установить скрипт из локального файла

## Provisioning

- Provisioning не повторяется после «Продолжить без скриптов» или после установки/удаления скрипта через менеджер

## Менеджер скриптов

- Заголовок экрана: «Менеджер скриптов» вместо «SBG Scout»
- Кнопка «Проверить обновления» отключается при пустом списке установленных скриптов

## UI

- Pull-tab настроек: увеличена ширина (13dp → 20dp) и прозрачность (alpha 0x80 → 0x40)

## Сеть

- HTTP-таймауты уменьшены: connect 10с, read 15с
- Дедлайн 15 секунд на получение ответа
