# v0.7

## Менеджер скриптов

- Интерфейс больше не залезает под статус-бар и кнопки навигации
- Прогресс загрузки отображается при установке конкретной версии и переустановке скрипта
- Тосты по завершении: загрузка пресета, обновление скрипта, проверка обновлений
- Корректное определение текущей версии в списке для скриптов, чей тег релиза не совпадает с `@version` (CUI в репо EUI)
