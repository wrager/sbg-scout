# v0.1

Тестовый релиз — базовая функциональность приложения.

## WebView

- Полноэкранный WebView с загрузкой SBG (`sbg-game.ru/app`)
- Immersive-режим (скрытие системных панелей)
- Поддержка cookies, remote debugging
- JS-бриджи для clipboard и share
- Логирование JS console в Logcat

## Менеджер юзерскриптов

- Парсер Tampermonkey-заголовков
- Хранилище скриптов с предустановленными правилами конфликтов
- Загрузка и автообновление скриптов
- Инжекция скриптов в WebView с учётом `@run-at`
- Обработка ошибок инжекции
- SVP загружается при первом запуске

## Платформа

- Runtime-разрешения на геолокацию
- Подписание release-сборки через CI
- CI: ktlint, detekt, unit-тесты, assembleDebug
- GitHub Actions workflow для автоматических релизов
