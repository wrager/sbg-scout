# v0.2

## Стартовый экран

- Новый экран-лаунчер со списком скриптов, тогглами включения/отключения и предупреждениями о конфликтах
- Добавление скриптов по URL через FAB-кнопку
- Удаление пользовательских (не предустановленных) скриптов
- Кнопка «Обновить все» — проверка и применение обновлений

## Меню скрипта

- Кнопка ⋮ на каждой карточке с действиями: выбор версии, переустановка, удаление
- Выбор конкретной версии для GitHub-скриптов через GitHub Releases API

## Настройки

- Экран настроек с переключателем автообновления скриптов и кнопкой «Сообщить об ошибке»
- Настройка полноэкранного (иммерсивного) режима

## Прочее

- Локализация: английский (по умолчанию) и русский
- Новая иконка приложения с глобусом SBG и текстом «USER SCRIPTS»
