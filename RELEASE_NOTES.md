# v0.15.1

## Новое

- Постфикс `SbgScout/<version>` в User-Agent — юзерскрипты могут определять, что запущены в SBG Scout, через `navigator.userAgent`.
- Подписи с кратким описанием для пресетных скриптов (SVP, EUI, CUI).
- Обнаружение изменений при переустановке скрипта с тем же номером версии — показывается кнопка reload game.
- Текст DEBUG на иконке debug-сборки.

## Улучшения

- Обновлён вшитый Vanilla+ до v0.8.1.
- Кнопка настроек Scout в игровых настройках — компактная строчка вместо отдельной секции.
- «Обновить все» применяет найденные обновления сразу, без повторной проверки.
- Скрипты без updateUrl пропускаются при проверке обновлений; кнопка проверки скрыта, если обновляемых скриптов нет.
- Компактнее layout карточек скриптов и footer настроек.
- Reload game вынесен в отдельный раздел «Game» в настройках.
- Сброс скролла при открытии настроек и менеджера скриптов.
